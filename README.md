
# Note App

### Dependencies
```
Docker >= 20.10.17
Docker-compose >= v2.6.0
```
## Init

Install docker and docker-compose and **then** run:
```
./init.sh
```
OR
```
sh init.sh
```

## Technologies

- Next.js (Typescript)
- Material UI
- NestJS
- TypeORM
- PostgreSQL
- Docker


## URL

- **FRONTEND:** http://localhost:3000

- **BACKEND:** http://localhost:3001

- **SWAGGER API DOCUMENTATION:** http://localhost:3001/docs
